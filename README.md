# Cleopas Strava

Cleopas Strava is a personal activity-visualization project that presents Strava exercise records as an interactive public archive. The site summarizes distance, activity type, yearly activity patterns, and route geometry in a browser-based map interface.

Live site: https://cleopas-strava.vercel.app

## Data Source

Activity data is synchronized from Strava through the Strava API and rendered as static site data for publication on Vercel. Imported activity names are preserved as provided by Strava.

## Purpose

This repository is maintained as a personal presentation and lightweight research artifact for reviewing long-term physical activity records, spatial movement patterns, and year-by-year training volume.
